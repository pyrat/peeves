h1. Protx VSP Server

h2. Introduction

Protx is an UK payment gateway. You can find more information on their VPS Server service "here":http://techsupport.protx.com/vspservercustom.asp.

h2. Setup and Installation

Edit <code>lib/peeves/config.rb</code> with your *vendor name*.

Test it by running
<pre>
<code>
ruby simulate.rb
</code>
</pre>
